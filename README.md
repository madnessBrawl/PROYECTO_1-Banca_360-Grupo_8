# PROYECTO_1--Banca360---Grupo_8
Proyecto #1 de la materia "Programación orientada a la web" - valor 30%(6 ptos) 
fecha de entrega: 14/5/2026

Objetivo del proyecto: Crear una pagina web que simule las funcionalidades de un banco, aplicando conceptos de manejo de
estado, validaciones de formularios y navegación dinámica.

divididos en:

(1) Módulo de autenticación y seguridad.
(2) Dashboard y visualización de datos.
(3) Operaciones bancarias.
(4) Requerimientos técnicos y UX

(!) ESTA PROHIBIDO USAR:
- Cualquier marco de trabajo (framework) o de desarrollo de software: React, Angular, Vue.js, Svelte, Solid.js, Next.js, Nuxt, js, entre otros.
- Librerías o preprocesadores de estilos: jQuery, Lodash, Axios, GSAP, Bootstrap, Tailwind CSS, Bulma, etc.).
- Todas las funciones, elementos y estilos deben implementarse utilizando JavaScript, HTML y CSS puros.

  
- OBSERVACIONES A TOMAR EN CUENTA -

- Se permite el uso de: HTML5, CSS3 y lenguaje JavaScript. Todas las funciones, elementos y estilos deben implementarse programando o maquetando con estas tecnologías puras.
- Solo debe desarrollarse la parte ejecutable en el navegador (interfaz del lado del cliente).
- Se utilizarán datos simulados y estáticos. No se desarrollará ninguna funcionalidad del lado del servidor ni bases de datos.
- Se debe entregar suministrando el enlace al repositorio en GitHub. Se sumará un (1) punto extra en caso de desplegar la página en un servidor web (recomiendo Github Pages).
- Como guía visual para los requerimientos, pueden tomar de base las distintas aplicaciones bancarias, sean Mercantil, Provincial o Bancamiga.
